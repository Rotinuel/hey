## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)

## Features Overview

### Hero Section
- Animated countdown timer to December 2025
- Call-to-action buttons (Get Tickets, Vote Now, Sponsor Us)
- Background image with overlay
- Responsive design

### Contest Showcase
- Cultural Carnival Showcase card
- Miss Abuja Detty December 2025 card
- Interactive participate buttons
- Trophy icons and modern card design

### Sponsors Section
- Abuja Continental Hotel
- Brave ICONS
- Kyros Automobile
- Clean logo presentation

### Events Preview
- Event cards with images
- Hover effects and animations
- Responsive grid layout

## Customization

The website uses a custom color palette defined in `tailwind.config.js`:
- Primary green colors for branding
- Forest green for accents
- Custom animations and transitions

## Deployment

The website is ready for deployment on platforms like:
- Vercel (recommended for Next.js)
- Netlify
- AWS Amplify
- Any static hosting service

Build the project with `npm run build` and deploy the `out` folder or use the platform's built-in deployment tools.
